# Документация: Календарь для совместного планирования

Добро пожаловать в документацию проекта **Sipi** — сервиса совместного планирования встреч.

Выберите раздел:

- [Начало работы](Getting-Started) — регистрация, вход, первичная настройка
- [Управление событиями](Events) — создание, редактирование и удаление личных событий
- [Совместное планирование](Collaboration) — создание встреч, поиск слотов, подтверждение участия
- [Настройки профиля](Settings) — рабочие часы, периоды недоступности, профиль пользователя
