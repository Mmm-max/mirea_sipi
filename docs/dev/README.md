# Документация разработчика — Sipi API

Документация генерируется из комментариев в коде инструментом **godoc** (стандартный инструмент Go).

## Как сгенерировать документацию

```bash
# Установить godoc (если не установлен):
go install golang.org/x/tools/cmd/godoc@latest

# Запустить локальный HTTP-сервер с документацией:
godoc -http=:6060

# Открыть в браузере:
# http://localhost:6060/pkg/sipi/internal/modules/
```

## Задокументированные пакеты

### `internal/modules/scheduling`

Содержит основную логику поиска и выбора временных слотов для встреч.

Ключевые функции:
- `Service.SearchSlots` — поиск доступных слотов с ранжированием
- `Service.SelectSlot` — подтверждение выбранного слота
- `Service.evaluateSlot` — оценка кандидатного временного интервала
- `withinWorkingHours` — проверка попадания в рабочие часы
- `overlaps` — проверка пересечения временных интервалов

### `internal/modules/meetings`

Управление жизненным циклом встреч.

Ключевые функции:
- `Service.CreateMeeting` — создание встречи с организатором как участником
- `Service.AddParticipants` — добавление участников с уведомлениями
- `Service.RespondInvitation` — ответ на приглашение

### `internal/modules/auth`

Аутентификация и управление сессиями.

Ключевые функции:
- `Service.Register` — регистрация с хешированием пароля Argon2id
- `Service.Refresh` — обновление токенов с защитой от повторного использования
